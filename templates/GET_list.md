# Template List

    GET services/:service_id/templates
    
Returns a list [Templates] scoped to the selected [Service].

### User Authorization Classes 
* account

## Parameters
None
### Sortable Fields
* subject
* display_name
* body

## Example
### Request

    GET https://api.zingle.me/v1/services/aff8bc93-6e28-4e70-8770-defa35cdfc1b/templates
