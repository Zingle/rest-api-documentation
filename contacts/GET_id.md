# Contact 

    GET services/:service_id/contacts/:contact_id
    
Returns a single [Contact] belonging to the specified service

## Parameters
None

## Example
### Request

    GET https://api.zingle.me/v1/services/aff7bc93-6e28-4e70-8770-defa35cdfc1b/contacts/89d9e8b5-b225-476c-a83e-50cdb720e3b1
